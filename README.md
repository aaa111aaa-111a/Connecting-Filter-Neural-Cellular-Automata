局所的な演算を繰り返す「ニューラルセルオートマトン」のアルゴリズムを用い、局所的な演算を繰り返す「ニューラルセルオートマトン」のアルゴリズムを用いることで、入力された幾何学図形を起点に、ピクセル同士が互いに影響を与え合い、全体がひとつの生き物のように連続的に変化する視覚表現を生み出しています。
入力は、カメラからの画像に加え、輝度重心を中心とする円です。
各ピクセルについて、自身の値・水平方向の勾配・垂直方向の勾配（Sobelフィルタによる）を周囲のピクセルから計算し、それらをランダムに設定した二層のニューラルネットワークに入力します。算出された変化量を直前のピクセルの値に加算することで状態を更新しています。Sobelフィルタにより各ピクセルが周囲の明暗の変化の方向と強さを知覚できるため、パターンが特定の方向へ向かって伝播し、波紋や渦のような有機的な広がりが生まれます。
